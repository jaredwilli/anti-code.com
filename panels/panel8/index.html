Webernote App

View App

This app is the result of a coding test I was asked to complete for an interview where I needed to build Evernote's desktop app as a web page, specifically using CSS to design it. I took it a step further than that.

  • I originally created for the interview test a functional note storing application, which mimics Evernote's UI. I used localStorage to store the data but decided that wasn't very scalable if I was going to actually make this thing worthwhile for me to make and function completely.

    I then rebuilt the JavaScript code from the ground up using a different style of coding. I decided to go with Firebase for storing the data and abstract away the backend coding so I could focus on the front-end. I modularized the JavaScript code and integrated user accounts via Twitter so others can use it and added the ability to add tags and notebooks to the notes, which will be used for filtering as well.

    This entire app works and functions like a desktop app would and maybe even better since some things just work more how you would like them to than you would expect, but you don't notice it since it just works. When you type in a note, it's saving and displaying that content in center column on keyUp. When you click a note, you can then edit it. When you add or remove tags, it updates the left nav menu. It communicates in 4 different directions using the Firebase API and some trickery on my part too.
