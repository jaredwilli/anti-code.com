This is a WordPress 3.0 custom built theme for a blog that made use of the new custom post type feature which was added in WordPress 3.0, before WordPress 3.0 was even released yet.

  • The site included the use of custom taxonomies, another newly added feature built into WordPress 3.0.

  • The design included a mega dropdown menu which was for displaying the custom taxonomies that were tied together with the custom post type for the blog.
