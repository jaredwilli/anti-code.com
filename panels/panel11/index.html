View Site

I made this as part of the interview process for a job with Google where they provided me with the text and told me to be creative and to try to get it done within 2 days if I could.

  • My Google Tour example has a similar look and feel as Google's UI and is 100% valid HTML5 semantic markup.

    They suggested I not use any libraries or frameworks and therefore I decided to use a function I wrote for a LinkedIn interview that to addClass() to Elements. I ended up creating a JavaScript micro library with it called Qlass.js, which I added removeClass() and hasClass() to compliment addClass. It takes both Elements and NodeLists as well.

  • This demo is coded with just HTML5, CSS3, and plain JavaScript. I tried out a couple different transition effects when navigating from page to page, one of which would zoom the visible page all the way out while zooming the next page that was clicked on in.

    I would've liked to have done a much cooler and more interactive Canvas solution for this to send back to them but I had determined that 2 days was not enough time to come up with something using Canvas unfortunately.
