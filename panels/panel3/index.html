HTML5 Drag and Drop Example

About

This is an example I made of how to implement HTML5 drag and drop features to make it possible to rearrange the images displayed on the right. You can click and drag them around to different places to organize them however you like.

  • This is a completely client-side example. No server-side code is used to store the new photo order.
  • While this example is using jQuery to do simple class adding and removing, the use of 3rd party libraries is not entirely needed. I made all the event handlers and DOM manipulations using straight conventional JavaScript.
  • This is example is functional in the browsers: Safari, Chrome, Firefox, and Internet Explorer 8 and 9.
